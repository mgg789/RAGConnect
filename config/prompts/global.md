## Memory system

You have access to a distributed memory system for user-level and project-level memory.

### Tools
| Tool | Purpose |
|------|---------|
| `memory_search` | Retrieve context before answering about a project. |
| `memory_write`  | Persist decisions, constraints, and outcomes immediately. |
| `memory_list_projects` | List available project destinations. |
| `memory_health` | Check reachability of all destinations. |
| `memory_graph` | Inspect graph payload for a destination. |
| `memory_entities` | List graph entities. |
| `memory_relations` | List graph relations. |
| `memory_documents` | List memory source documents. |
| `memory_ingest_bulk` | Insert multiple records in one operation. |
| `memory_rebuild_index` | Trigger graph/index rebuild. |
